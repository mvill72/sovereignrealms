## Technical Deep-Dive: The Daimon in the Architecture

### Farcaster's Hybrid Architecture (2026)

```
On-Chain (Optimism L2)
├── IdRegistry - Maps FID to Ethereum custody address
├── StorageRegistry - Meters data storage (~$7/year)
└── KeyRegistry - Manages EdDSA delegated signing keys

Off-Chain (Snapchain)
├── BFT Consensus (Malachite, Tendermint-derived)
├── Account-Level Sharding
├── ~10k+ TPS for social data
└── Verifiable but public data layer

Client Layer
├── Warpcast (flagship)
├── Frames (interactive mini-apps)
└── Multiple open clients
```

**Philosophy**: Sufficient decentralization. Public-by-design. Optimize for UX and engagement while maintaining verifiability.

**Shadow**: The psyche is externalized before it is internalized. Engagement precedes reflection.

### SovereignRealm's Local-First Architecture (2026)

```
Browser Vault (Private-by-Default)
├── IndexedDB - Primary local storage
├── Web Crypto API - Native encryption (SHA-256, AES-GCM)
├── OPFS - Future-proof enhanced performance
└── One-click JSON export - Freedom to leave

Four Sovereign Circles (Enforced Locally)
├── 🔒 Vault Only - Never shared, never uploaded
├── 👨‍👩‍👧 Family Realm - E2E encrypted, CircleKey gated
├── 💼 Work Collegium - Professional persona, gated
└── 🌐 Outer World - Public, optionally federated

Optional On-Chain Layer (Any EVM Chain)
├── SovereignProfile (ERC-721) - Identity NFT
├── CircleKeys (ERC-1155) - Access tokens
└── Multi-chain portability (Mainnet, Polygon, Base, Arbitrum, Optimism)

Optional Federation Layer
├── ActivityPub (Fedify) - True Fediverse bridge
├── WebFinger - Actor discovery
└── Opt-in only for Outer World posts
```

**Philosophy**: Local-first as sacred architecture. Private-by-default. Individuation before participation.

**Light**: The Vault is built first. The Self confronts its shadow in private before any outer projection.
